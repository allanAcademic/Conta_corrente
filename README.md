# 💰 Conta Corrente - Banco Digital

Sistema simples de gerenciamento de conta bancária desenvolvido em Java como exercício prático de lógica de programação.

## 📋 Descrição

Um programa de terminal que simula operações bancárias básicas:
- Consulta de saldo
- Depósitos (receber valor)
- Transferências (com validação de saldo)
- Menu interativo com loop

## 🚀 Funcionalidades

- ✅ Consultar saldo atual
- ✅ Receber valores (depósito)
- ✅ Transferir valores (com verificação de saldo)
- ✅ Tratamento de opções inválidas
- ✅ Interface via terminal

## 🛠️ Tecnologias

- Java 11+
- Git & GitHub

## ▶️ Como executar

### Pré-requisitos
- Java JDK 11 ou superior instalado

### Passos

```bash
# Clone o repositório
git clone https://github.com/allanAcademic/Conta_corrente.git

# Entre na pasta do projeto
cd Conta_corrente

# Compile o código
javac src/Main.java

# Execute o programa
java src/Main
